

A mobile app for browsing and buying yarn, built as a college project using Kotlin and Android Studio.

![Platform](https://img.shields.io/badge/Platform-Android-3DDC84?style=flat-square&logo=android&logoColor=white)
![Language](https://img.shields.io/badge/Language-Kotlin-7F52FF?style=flat-square&logo=kotlin&logoColor=white)
![Status](https://img.shields.io/badge/Status-In%20Development-orange?style=flat-square)

---

- Login and signup
- Browse yarn products
- Add to cart and checkout
- Clean and simple UI


- Kotlin
- Android Studio
- MVVM Architecture

1. Clone the repo
```bash
git clone https://github.com/muneera7800-boop/nuve-yarn.git
```
2. Open in Android Studio
3. Hit Run ▶️

Built by a college student learning Android development. Still a work in progress — feel free to explore the code!
